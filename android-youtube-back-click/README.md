# YouTube Back Click (Android MVP)

개인 사용 목적의 Android MVP 입니다.
오버레이 버튼을 눌러 유튜브 플레이어의 왼쪽 영역 더블탭 제스처를 실행합니다.

## 기능

- 앱 위 오버레이 버튼 1개
- 버튼 클릭 시 접근성 서비스로 더블탭 제스처 실행
- 탭 실행 횟수(1~3) 설정
- 제스처 좌표 비율(X/Y) 설정

## 프로젝트 열기

1. Android Studio 실행
2. `android-youtube-back-click` 폴더 열기
3. Gradle Sync 실행

## 실행 순서

1. 앱 실행
2. `오버레이 권한 열기`에서 권한 허용
3. `접근성 설정 열기`에서 `YouTube Back Click Service` 활성화
4. `오버레이 시작` 누르기
5. 유튜브 앱에서 플로팅 버튼 클릭

## 주의

- 앱/기기/화면비에 따라 탭 좌표를 조절해야 합니다.
- 유튜브 UI 업데이트 시 좌표 재보정이 필요할 수 있습니다.
- 본 코드는 개인 실험용 MVP입니다.
